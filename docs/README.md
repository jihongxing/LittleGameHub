1. 确定文档类型和分类
2. 按照[文档命名规范](./GameHub_文档管理规范.md#2-文档命名规范)命名文档
3. 使用[文档模板](./GameHub_文档管理规范.md#5-文档模板)创建文档
4. 添加文档元数据
5. 提交审查
6. 更新相关索引

### 如何更新现有文档

1. 修改文档内容
2. 更新文档版本和元数据
3. 提交审查
4. 更新相关索引
