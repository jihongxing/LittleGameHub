# 🎮 小游戏聚合平台前后台功能明细（含裂变功能）

---

## 一、前端功能明细

### 1. 游戏列表页（首页）
- 分类标签：推荐 / 热门 / 最新 / 个性化推荐
- 游戏卡片显示：封面图 + 名称 + 标签 + 积分奖励 + 好友玩过标识
- 点击游戏卡片：进入游戏详情页
- 游戏详情页：游戏封面、简介、开始按钮、可获得积分、广告提示、好友评价
- 游戏开始：iframe 加载游戏，计时、积分计算
- 下拉刷新 & 加载更多
- 顶部导航：搜索框、积分入口、个人中心快捷入口
- 底部 Tab 导航：首页 / 积分 / 会员 / 我的 / 社交
- **个性化推荐区**：基于用户画像和游戏行为推荐
- **场景化推荐**：按"通勤"、"课间"、"睡前"等场景分类
- **社交推荐**：显示"好友正在玩"和"热门游戏"
- **收藏管理**：游戏收藏、收藏分类、收藏同步
- **离线下载**：游戏下载、下载管理、离线游戏库
- **游戏更新**：更新提醒、自动更新、更新历史

### 2. 积分页（任务与兑换中心）
- 显示当前用户积分
- 积分任务区：每日签到、看广告、玩游戏、分享好友
- 积分兑换区：会员体验卡、现金红包、虚拟道具
- 点击任务：触发广告或小游戏，完成后积分到账
- 点击兑换：二次确认，积分扣除，生成兑换记录
- 积分排行榜展示

### 3. 会员页
- 显示会员状态及到期时间
- 会员特权展示：去广告、积分加成、云端存档、抢先体验、新游戏优先
- 会员购买：Web 支付、Android 支付、iOS IAP
- 支付成功后权限更新、积分加成同步

### 4. 我的页面（用户中心）
- 头像、昵称、积分、会员状态
- 我的游戏记录
- 积分兑换记录
- 成就徽章展示
- 设置中心：清缓存、意见反馈

### 5. 游戏内互动
- 积分奖励动画
- 分享邀请好友按钮
- 排行榜实时显示
- **游戏进度云同步**：跨设备保存游戏进度
- **成就系统**：游戏内成就与平台积分联动
- **好友互动**：游戏内好友对战、协作功能
- **实时消息**：好友游戏邀请、成绩分享

### 6. 广告与激励
- 激励视频广告、插屏广告、Banner 广告
- 广告完成事件回调给前端，用于积分发放或游戏奖励

### 7. 裂变与邀请功能（新增）
- 邀请好友注册 / 玩游戏获得奖励
- 邀请链接 / 二维码生成与分享
- 好友完成任务后邀请者可获得积分或道具
- 裂变排行榜展示（邀请人数排名、奖励排行榜）
- 裂变活动弹窗或首页展示
- 裂变任务完成动画与奖励提示
- **社交分享优化**：一键分享到微信、QQ等社交平台
- **邀请进度追踪**：实时显示邀请进度和奖励状态
- **团队邀请**：支持团队邀请模式，共同完成任务  

---

## 二、后台功能明细

### 1. 游戏管理模块
- 游戏上架/下架/编辑
- 游戏灰度发布（按用户分组/地区/渠道）
- 积分策略配置（每款游戏奖励积分、最低游戏时长）
- 广告策略配置（激励广告、插屏、Banner 显示策略）
- 游戏分类、标签管理
- 游戏版本管理与回滚
- 游戏统计：用户播放次数、平均时长、广告点击率、积分发放统计
- **个性化推荐配置**：基于用户行为的游戏推荐算法设置
- **场景化游戏管理**：按场景分类管理游戏展示
- **游戏内容审核**：确保游戏内容合规安全
- **开发者管理**：游戏开发者入驻与收益分成管理
- **收藏游戏管理**：用户收藏游戏列表、收藏分类管理
- **离线下载管理**：游戏资源打包、下载队列管理、存储空间管理
- **游戏更新管理**：版本控制、增量更新、自动更新策略

### 2. 用户管理模块
- 用户列表与信息查看
- 用户积分统计与流水查询
- 用户会员状态与购买记录查询
- 用户行为监控（异常行为、作弊检测）
- 邀请好友数据管理：邀请人数、奖励发放状态、裂变链路统计
- **用户画像系统**：基于行为数据的用户标签管理
- **用户行为分析**：游戏偏好、活跃时段、留存分析
- **用户成长体系**：等级、成就、徽章管理
- **用户反馈管理**：意见收集与处理流程

### 3. 积分系统管理
- 积分任务配置：签到、观看广告、分享、试玩、邀请好友
- 积分兑换配置：会员、红包、虚拟道具
- 积分发放与扣除记录管理
- 防刷规则配置（频次限制、IP/设备限制）
- 裂变奖励发放规则配置（邀请完成奖励、限制领取次数）

### 4. 会员管理模块
- 会员等级/套餐管理
- 会员购买记录查询与续费管理
- 权限开关（去广告、积分加成等）
- 支付验证日志管理（Apple IAP / 微信 / 支付宝）

### 5. 广告管理模块
- 广告位管理（激励视频、插屏、Banner）
- 广告投放策略配置（优先级、频次、地域）
- 广告统计报表（曝光量、点击率、收入）
- 多广告源接入与切换

### 6. 裂变管理模块（新增）
- 裂变活动管理：创建 / 编辑 / 下线
- 邀请奖励配置：积分、道具、会员时长
- 裂变数据统计：邀请人数、奖励发放、链路有效性
- 异常裂变检测：防刷、重复注册、虚假邀请

### 6.1 社交功能管理模块（新增）
- 好友系统管理：好友关系、好友推荐、好友互动
- 实时排行榜管理：游戏排行、积分排行、邀请排行
- 社群管理：游戏社群创建、管理、活动发布
- 社交内容审核：用户生成内容、聊天内容审核
- 社交数据统计：互动频次、分享转化率、社交留存
- 团队活动管理：团队任务、团队竞赛、团队奖励

### 7. 数据统计与分析
- 日活/月活用户统计
- 游戏排行榜数据
- 积分消耗与充值统计
- 广告收益统计
- 会员收入统计
- 裂变活动效果分析（邀请转化率、奖励领取率）
- **用户行为分析**：游戏偏好、活跃时段、留存分析
- **推荐系统效果分析**：点击率、转化率、用户满意度
- **社交功能分析**：好友互动频次、社群活跃度、分享转化率
- **A/B测试管理**：功能测试、界面测试、算法测试
- **数据可视化看板**：实时数据监控、趋势分析、异常告警

### 8. 系统与运维
- 登录权限控制（管理员/运营/审核）
- 操作日志与审计
- 系统配置管理（参数配置、版本发布）
- 灰度发布与回滚管理

### 9. 安全与风控
- 异常行为监控（刷分、异常充值、作弊）
- 用户设备/IP 风控设置
- 关键操作二次确认（积分扣除、会员退款）
- 数据备份与恢复机制
- 裂变防刷规则（限制同 IP / 设备邀请奖励、重复注册检测）
- **内容安全审核**：游戏内容、用户生成内容、社交内容审核
- **数据隐私保护**：用户数据加密、权限管理、隐私合规
- **反作弊系统**：游戏行为分析、异常检测、实时拦截
- **安全审计**：操作日志、安全事件记录、合规检查
- **离线内容管理**：离线游戏资源加密、离线数据同步、离线使用权限控制
